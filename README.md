# Traveloop

Traveloop is a smart and user friendly multi city travel planning platform that helps users organize trips, manage destinations, schedule activities, track budgets, and share itineraries through a clean and responsive interface.

**Repository:** [github.com/sanjay-offl/traveloop](https://github.com/sanjay-offl/traveloop)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Main routes include `/` (auth entry), `/home` (marketing), `/dashboard` (planner), `/login`, and `/register`.
